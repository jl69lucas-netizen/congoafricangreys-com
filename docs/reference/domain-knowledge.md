# African Grey Domain Knowledge

## Two Variants (= MFS size tiers equivalent)

| Variant | Full Name | Size | Price Range | Notes |
|---|---|---|---|---|
| CAG | Congo African Grey | Larger, ~400–600g | $1,500–$3,500 | Most common, red tail |
| TAG | Timneh African Grey | Smaller, ~275–375g | $1,200–$2,500 | Darker grey, maroon tail |

## Key Trust Signals

- USDA Animal Welfare Act license
- CITES captive-bred documentation
- Avian vet health certificate
- DNA sexing certificate
- Weaning status (fully weaned vs still weaning)
- Hand-fed / hand-raised (socialization)
- Hatch certificate / band/ring number

## Key Health Conditions to Reference

- Psittacine Beak and Feather Disease (PBFD)
- Proventricular Dilatation Disease (PDD)
- Aspergillosis (respiratory)
- Calcium/Vitamin A deficiency
- Feather destructive behavior
