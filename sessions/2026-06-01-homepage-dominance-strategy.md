## 8. AEO / GEO — "Be the source AI chatbots cite"

AI engines (ChatGPT, Claude, Gemini, Perplexity, Google AIO, Bing Copilot) cite sources that are: **(a) entity-clear, (b) declaratively factual, (c) schema-structured, (d) corroborated by authority links, (e) fresh.** Action plan for the homepage:

1. **Declarative answer blocks** — each H2 question must be followed by a ≤320-char, self-contained factual answer (so it can be lifted into an AI Overview / featured snippet). Most already are; tighten the price, lifespan, legality answers.
2. **FAQPage schema** — expand to cover: price, lifespan, legality (CITES), Congo vs Timneh, talking ability, food, shipping. ✅ exists; add lifespan + legality + food Q&As.
3. **Entity corroboration** — `sameAs` Organization schema + Person schema + IUCN/CITES/Pepperberg outbound links = the citation triangulation AI models trust.
4. **"Stat-forward" facts** — AI loves liftable numbers: "40–60 year lifespan," "100–1,000 word vocabulary," "$1,500 floor," "12+ years," "Appendix I since Jan 2017." Counter snippet does this ✅.
5. **Bing/DuckDuckGo/Yahoo** — DuckDuckGo + Yahoo are powered by **Bing's index**. Winning Bing = winning all three. Submit the homepage to **Bing Webmaster Tools** + **IndexNow** (CAG already has IndexNow per deploy-verifier). Bing weights exact-match + schema + on-page heading match heavily — the head-term fix (§1) helps here too.
6. **Freshness signal** — add a visible "Updated [Month Year]" + `dateModified` in schema; AI engines prefer recently-updated sources.
7. **Get cited elsewhere** — AI models cite what the web cites. The `cag-backlink-outreach-agent` (resource pages, avian-vet referrals) + `cag-directory-submission-agent` feed this.
